CSS 색상

CSS 색상을 표현하는 방법은 색상명을 쓰는것과 16진수 표현법(hexadecimal code)이 있습니다.

색상명

색상명을 씁니다.

.test {
    background-color: red;
}

16진수 표현법(hexadecimal code)

16진수를 사용하여 색상을 표현합니다.

.test {
    background-color: #FF0000;
}

위의 코드의 FF0000이 뜻하는 것은 red: FF, green: 00, blue: 00을 16진수로 나타낸 것을 표현한 것입니다. 00이 가장 작은 수, FF가 가장 큰 수 이고 큰 숫자가 될 수록 밝은 색을 나타냅니다.
