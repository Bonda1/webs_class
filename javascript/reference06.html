조건문

조건문은 조건식의 값이 참(true)인지, 거짓(false)인지에 따라 자바스크립트 코드를 제어합니다.

if문

if문은 조건식을 만족(true)할 경우에만 코드를 실행합니다. 다음은 if문의 기본형입니다. 조건식은 Boolean() 내장 메서드와 마찬가지로 그 어떤 데이터를 입력해도 true 또는 false를 반환합니다.

if(조건식){
    자바스크립트 코드;
}
적용 예제1
var num=10;
    if(num<(500){ //true를 반환합니다.
        document.write("hello");
}
