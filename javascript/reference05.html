연산자

연산자란 프로그래밍에서 쓰이는 기호들이며, 수나 식을 일정한 규칙에 따라 계산하는 것을 의미한다.

예를 들어 나의 평균 체중을 구할 때 빼고 곱하는 작업 등은 산술 연산자 이용한다. 그리고 빼기, 더하기, 곱하기, 나누기, 비교 등을 하는 작업을 연산 작업이라고 한다.

자바스크립트 프로그래밍에서 사용하는 연산자

  • 산술
  • 문자 결합
  • 대입(복합 대입)
  • 증감
  • 비교
  • 논리
  • 삼항 조건

문자 결합 연산자

제목 그대로 여러 문자를 결합하여 하나의 문자열을 만들 때 사용한다. (+)연산자를 사용한다.

<script>
    var num1 = "Hello";
    var num2 = " World";

    result = num1 + num2;   //Hello World

    document.write(result);
</script>

대입 연산자

(=)연산자를 사용하여 변수에 값을 대입합니다.

<script>
    var num1 = "Hello"
    var num2 = "World"
            
    result = num1 + num2;   //Hello World
            
    document.write(result);
</script>
