변수란?

설명과 같이 변수는 저장된 값을 식별할 수 있는 이름입니다. 변수의 값을 저장하는 것을 할당 이라고 하고 그 값을 읽어오는 것을 참조 라고 합니다. 여기서 변수 이름을 식별자 라고도 합니다. 주의할 점은 식별자는 값이 아닌 메모리의 주소를 기억하고 있습니다. 

var result(변수 이름(식별자)) = 10 + 20;

변수 선언 방법

변수 를선언하는 방법은 var 키워드를 변수명 앞에 붙입니다. (현재는 'let' 이라는 키워드를 주로 사용합니다.). 변수명에는 한글을 사용할 수 없고, 영문과 숫자 그리고 일부 특수 문자[밑줄(_), 달러기호($)]만 포함할 수 있습니다. 

var 변수명 = 값;
var 변수명 = 값;

let 변수명 = 값;
let 변수명 = 값;

변수명

자바스크립트에는 의미를 가진 단어들이 있기 때문에 사용자가 임의로 사용할 수 없는 단어들이 있습니다.

키워드 : 자바스크립트에서 사용하는 단어
식별자 : 사용자가 임의로 사용하는 단어

식별자를 만드는 규칙

  • 사용자가 사용할 수 있는 문자: 알파벳(a~z, A~Z), 숫자(0~9), 밑줄(_), 달러기호($)
  • 첫 글자는 숫자를 사용할 수 없습니다.
  • 공백을 사용할 수 없습니다. : 밑줄(_) 사용가능, 하이픈(-) 사용안됩니다.
  • 특수기호를 사용할 수 없습니다. : 달려기호($)는 사용가능합니다.
  • 키워드를 사용할 수 없습니다.

단어를 조합할 때 규칙

  • 언더스코어 노테이션 : (color_top) : 중간에 '_'를 사용합니다.
  • 캐멜 노테이션 : (color_Top) : 두번째 단어의 첫 글자를 대문자로 사용합니다.
  • 파스칼 노테이션 (ColorTop) : 모든 단어의 첫 글자를 대문자로 사용합니다.
