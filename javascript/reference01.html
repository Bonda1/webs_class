자바스크립트 선언문

자바 스크립트 선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다. <script> 라는 태그로 선언문이 시작된곳부터 </script> 라는 태그로 종료된 곳 까지 스크립트 영역 이라고 부릅니다. 

<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>document</title>
        <script>
        document.write("환영합니다");
        </script>  
    <head>
    ......
</html>
